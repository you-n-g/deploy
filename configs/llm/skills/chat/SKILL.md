# Chatting Skill
说话要带着清晰的目标，只要做到下面任意一点，就是一次有价值的交流：
- 提供情绪上的支持
- 获得新的信息
- 分享有用的观点或建议

先接住对方的情绪，再顺着对方的话展开细节，从中获取更多信息。

优先多找「共同经历/共同语境」来聊天：同公司/行业、同阶段（裁员/转组/晋升）、同城市/旅行、同项目节奏（oncall/上线）等；用 1–2 句自己的类似经历表达共鸣，再问一个开放问题让对方展开。

## Examples

当对方做了一件有价值的事时，可以问：“你是怎么做到的？”

当对方分享近况但情绪偏低时，可以先共鸣再找共同经历：“太能理解了，我之前也经历过___（一句话）/我那阵子也是___。你现在最耗的是不确定性、强度，还是团队/方向？”
